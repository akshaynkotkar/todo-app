# Setup Steps
1. **Clone the repository:**
    ```bash
    git clone https://github.com/akshaynkotkar/todo-app.git
    cd todo-app
    ```
## Backend Setup

1. **Initialize the project:**
    ```bash
    cd backend
    npm install
    ```

2. **Install dependencies:**
    ```bash
    npm install express cors mongoose dotenv
    ```

3. **Start the Server**
    ```bash
    npm start
    ```

## Frontend Setup

1. **Initialize the project:**
    ```bash
    cd frontend
    ```

2. **Install dependencies:**
    ```bash
    npm install
    ```

3. **Start the development server:**
    ```bash
    npm run dev
    ```
